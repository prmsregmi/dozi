# Transcription Agent

LiveKit agent for real-time audio transcription. Supports Deepgram Nova-3 (streaming, default) and OpenAI gpt-4o-transcribe (batch fallback).

## Local Development

Install dependencies:

```bash
pip install -r requirements.txt
```

Create `.env` in `agents/` directory:

```bash
DEEPGRAM_API_KEY=...
OPENAI_API_KEY=sk-...
LIVEKIT_URL=wss://your-project.livekit.cloud
LIVEKIT_API_KEY=...
LIVEKIT_API_SECRET=...
```

Run locally:

```bash
python transcription_agent.py dev
```

## Deploy to LiveKit Cloud

Install LiveKit CLI:

```bash
brew install livekit-cli
```

Login and deploy:

```bash
lk cloud auth
cd agents/
lk agent create --project dozi
```

Set `DEEPGRAM_API_KEY` (and optionally `OPENAI_API_KEY`) in LiveKit Cloud dashboard under **Settings → Agents**.

LiveKit credentials are automatically provided by the platform. The `livekit.toml` file is auto-generated by the LiveKit CLI and contains your project config. It's gitignored for security.
